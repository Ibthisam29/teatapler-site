How to Get Your Keys
1. Go to dashboard.stripe.com → Create free account
2. Left sidebar → Developers → API Keys
3. Copy your Publishable key (starts with pk_)

1. Create a Payment Link in your Stripe dashboard → it gives you a buy.stripe.com/xxx URL
2. Paste the URL below next to the matching product
3. Click Save All Links — every button on your website updates instantly
4. Customers click Reserve → land on Stripe's hosted checkout → pay securely → you get notified

Create Links in Stripe

        

          
1. Go to dashboard.stripe.com/payment-links

          
2. Click + New

          
3. Add product → name, price, image, description

          
4. Set quantity → enable Collect customer email

          
5. In After payment → add confirmation message

          
6. Copy the buy.stripe.com/xxx URL

          
7. Paste it above → Save All Links

          

            Create one link per product. Use the same Stripe account linked to your pk_live_ key.
